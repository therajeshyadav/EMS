# Employee Management System

The **Employee Management System (EMS)** is a web application that helps organizations manage employees and their daily operations efficiently.  
It provides two main roles: **Admin** and **Employee**, with distinct functionalities tailored to their needs.

---

## 🚀 Features

### 👩‍💼 Employee Module
- **Authentication & Profile**
  - Employee login/signup
  - View and update profile
- **Attendance**
  - Mark daily attendance
  - View attendance history
- **Tasks**
  - View assigned tasks
  - Accept tasks
  - Update task status (Completed / Failed)
- **Payments**
  - View payment slips
- **Notifications**
  - Receive updates/announcements from Admin
- **Leave Management**
  - Apply for leave
  - Check leave status (Approved/Rejected)
  - View leave history

---

### 🛠️ Admin Module
- **User Management**
  - Add new employees
  - Remove employees
  - View employee profiles and activity
- **Attendance Management**
  - View attendance of all employees
- **Task Management**
  - Assign tasks to employees
  - Monitor task progress (Completed / Failed)
- **Leave Management**
  - Review employee leave requests
  - Approve or reject applications
- **Payment Management**
  - Generate and manage employee payment slips
- **Notifications**
  - Send announcements or notifications to employees

---

## 🏗️ Tech Stack

- **Frontend:** React.js / HTML / CSS / JavaScript 
- **Backend:** Node.js / Express.js 
- **Database:**  MongoDB 
- **Authentication:** JWT 
- **Deployment:** frontend at Vercel and Backend at Render
